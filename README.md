# 🎯 ExamHub — Online Mock Test Platform

ExamHub is a full-stack web application designed to help students prepare for competitive exams through interactive mock tests, performance analytics, and a user-friendly dashboard. The platform provides a real exam-like experience with secure authentication and scalable exam management.

---

## 🚀 Live Demo

🔗 https://examhub.infinityfree.me/

## ✨ Features

* 🔐 User Authentication (Login / Signup / Google OAuth)
* 📝 Interactive Mock Test Interface

* 👤 User Dashboard

```
│── users/        # User dashboard

```

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/vansh9676/examhub.git
```

2. Move project to your server directory (htdocs / public_html)

3. Create a MySQL database

4. Update database credentials in:

```
config.php
```

5. Import the SQL file (if provided)

6. Run the project in browser:

```
http://localhost/examhub
```

---

## 🔑 Environment Configuration

Update database credentials:

```
DB_HOST=localhost
DB_USER=root
DB_PASS=yourpassword
DB_NAME=examhub
```
